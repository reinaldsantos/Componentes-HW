BIOS/UEFI

O que é BIOS?

BIOS (Basic Input/Output System) é um firmware armazenado em um chip na placa-mãe que inicializa e testa os componentes de hardware durante a inicialização do computador e carrega o sistema operacional.

BIOS vs UEFI

  • BIOS Legado:
    • Interface mais antiga e limitada
    • Suporte apenas para MBR
    • Limitado a 2TB de armazenamento
    • Interface baseada em texto
  • UEFI (Unified Extensible Firmware Interface):
    • Interface moderna e gráfica
    • Suporte para GPT
    • Sem limitação de armazenamento
    • Recursos avançados de segurança
    • Inicialização mais rápida

Funções Principais

  • POST (Power-On Self-Test)
    • Verifica CPU
    • Testa memória RAM
    • Identifica dispositivos conectados
    • Verifica integridade do sistema
  • Configurações de Hardware
    • Velocidade do processador
    • Configurações de memória
    • Ordem de boot
    • Gerenciamento de energia
