Tipos de Memória

Memória RAM (Random Access Memory)

A memória RAM é um tipo de memória volátil que armazena temporariamente dados e programas em uso pelo computador. É essencial para o funcionamento do sistema e execução de programas.

Características da RAM

  • Memória volátil (perde dados quando desligada)
  • Alta velocidade de acesso
  • Leitura e escrita rápidas
  • Acesso aleatório aos dados

Tipos de RAM

  • DDR3 - Terceira geração
  • DDR4 - Quarta geração, mais rápida e eficiente
  • DDR5 - Última geração, maior velocidade e capacidade

Memória ROM (Read Only Memory)

A memória ROM é um tipo de memória não-volátil que mantém seu conteúdo mesmo quando o computador está desligado. É utilizada para armazenar instruções permanentes do sistema.

Características da ROM

  • Memória não-volátil (mantém dados quando desligada)
  • Somente leitura (não pode ser modificada facilmente)
  • Mais lenta que a RAM
  • Armazena o BIOS/UEFI do computador

Tipos de ROM

  • ROM tradicional (apenas leitura)
  • PROM (Programmable ROM - programável uma única vez)
  • EPROM (Erasable PROM - pode ser apagada e reprogramada)
  • EEPROM (Electrically EPROM - reprogramável eletricamente)
  • Flash ROM (tipo mais moderno, usado em BIOS/UEFI)

Comparação RAM vs ROM

  • RAM:
    • Volátil
    • Maior velocidade
    • Leitura e escrita
    • Uso em programas e dados temporários
  • ROM:
    • Não-volátil
    • Menor velocidade
    • Principalmente leitura
    • Uso em instruções permanentes

Especificações Técnicas

  • Velocidade de acesso
  • Capacidade de armazenamento
  • Latência
  • Frequência de operação
  • Tensão de operação
  • Tecnologia de fabricação
