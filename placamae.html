Placa-Mãe

O que é uma Placa-Mãe?

A placa-mãe é o componente principal que conecta e permite a comunicação entre todos os outros componentes do computador.

Principais Componentes

  • Socket do processador
  • Slots de memória RAM
  • Slots PCIe
  • Conectores SATA
  • Chipset
  • BIOS/UEFI

Formatos (Form Factors)

  • ATX
  • Micro-ATX
  • Mini-ITX
  • E-ATX

Recursos Comuns

  • Suporte a overclock
  • Áudio integrado
  • Rede integrada
  • USB ports
  • RGB headers
  • M.2 slots

Principais Fabricantes

  • ASUS
  • MSI
